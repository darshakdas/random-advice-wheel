# 🎡 Random Advice Wheel (Angular)

A fun Angular mini project inspired by spinning‑wheel interactions.
Click the button to spin the wheel and receive a random piece of advice.

## Features
- Animated wheel spin
- Random advice selection
- Clean Angular component architecture
- CSS‑based transitions

## Tech Stack
- Angular
- TypeScript
- HTML / CSS

## Purpose
Built as a creative side project to demonstrate Angular fundamentals,
event handling, state management, and playful UI interactions.
